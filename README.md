## 🥞 编译运行

* 本项目基于xmake构建
* 请确保你的环境中有 `gcc`命令
* 请确保在项目根目录下执行命令

``` bash
xmake
./build/crepl
```

## 🔧 运行测试

* 请确保你的环境中有 `python3` 以及相关的库
* 请确保在项目根目录下执行命令

``` bash
./run_test.py
```
